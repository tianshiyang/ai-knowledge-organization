## 简介

屏蔽 OpenClaw CLI 横幅消息。当用户说 "hide openclaw banner"、"disable openclaw banner"、"suppress openclaw banner" 等时使用。

## 适用场景

- 隐藏 OpenClaw CLI 启动横幅
- 减少 CLI 输出噪音
- 自动化脚本中屏蔽 banner

## 核心功能

- 屏蔽 OpenClaw CLI 横幅消息
- 减少终端输出
- 自动化友好

## 安装方式

```bash
clawhub install shut-up-openclaw-cli
```
