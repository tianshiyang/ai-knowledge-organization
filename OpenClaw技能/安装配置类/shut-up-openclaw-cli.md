## 一句话定位

一个非常小但很实用的 CLI 降噪技能，用于隐藏 OpenClaw 启动横幅，特别适合自动化和脚本场景。

## 适合谁

- 经常在终端里反复调用 OpenClaw 的用户
- 做自动化脚本、日志采集或批处理的人
- 讨厌无关输出噪音的 CLI 用户

## 典型使用场景

- 隐藏 OpenClaw CLI 启动横幅
- 减少 CLI 输出噪音
- 自动化脚本中屏蔽 banner

## 为什么值得用

- 它虽然小，但对高频 CLI 用户的体验提升很直接。
- 在脚本和自动化日志里，减少无关输出可以让结果更干净。

## 前端视角说明

如果你把 OpenClaw 当成本地开发工具频繁调用，这种降噪类技能实际会比很多“大而全”技能更高频。

## 通用性说明

通用性一般，但在“高频 CLI 使用”这一类场景中很实用。

## 风险与注意事项

- 它只解决输出体验问题，不提升 OpenClaw 功能本身。
- 更适合放在个人习惯优化工具链里，而不是团队标准能力列表里。

## 安装方式

```bash
clawhub install shut-up-openclaw-cli
```
