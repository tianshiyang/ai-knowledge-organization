## 简介

跨平台操作和故障排除 OpenClaw CLI 的全面技能，涵盖安装、Gateway/Node 生命周期、渠道登录、消息收发、Agent 轮次、模型、插件和系统健康检查等全部环节。

## 适用场景

- OpenClaw CLI 日常使用
- Gateway 和 Node 生命周期管理
- Channel 登录和配置
- 消息收发调试
- Agent 轮次管理
- 模型配置和切换
- 插件管理
- 系统健康检查
