## 简介

为 OpenClaw 安装和配置 Owletto 记忆插件，包括 OAuth 登录和 MCP 健康验证。

## 适用场景

- 需要为 OpenClaw 添加记忆功能
- Owletto 插件安装
- OAuth 认证配置
- MCP 健康检查

## 核心功能

- Owletto 插件安装
- OAuth 登录配置
- MCP 健康验证
- 记忆功能集成

## 安装方式

```bash
clawhub install owletto-openclaw
```
