## 简介

OpenClaw 运维防呆与排障标准化技能。用于跨设备（Gateway + Mac nodes）巡检、远程执行稳定性治理、CLI-only 兼容、配对/审批异常排查、以及对外发布前脱敏检查。

## 适用场景

- 遇到 "又报错了/审批超时/pairing required/system.run failed" 等运维问题
- 需要标准化运维规则和巡检流程
- 跨设备运维和远程执行场景

## 核心功能

- 跨设备（Gateway + Mac nodes）巡检
- 远程执行稳定性治理
- CLI-only 兼容性处理
- 配对/审批异常排查
- 对外发布前脱敏检查

## 安装方式

```bash
clawhub install openclaw-ops-guardrails
```
