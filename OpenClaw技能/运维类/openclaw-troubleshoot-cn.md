## 一句话定位

一个偏中文语境的 OpenClaw 故障排查技能，适合快速诊断常见问题和理解排障步骤。

## 适合谁

- 遇到 OpenClaw 运行问题的用户
- 更习惯中文排障说明的人
- 做日常巡检或问题定位的维护者

## 典型使用场景

- 遇到 OpenClaw 运行时问题
- 需要故障诊断指导
- 运维人员日常巡检

## 为什么值得用

- 如果你不想先看一整套英文或综合运维资料，这类中文排障技能更容易快速上手。
- 适合把“先定位问题”这一步单独做好。

## 前端视角说明

对偶尔使用 OpenClaw 的前端用户也友好，因为它比综合运维技能更轻量。

## 通用性说明

只适用于 OpenClaw 故障排查，不是通用技能。

## 风险与注意事项

- 更适合常见问题和标准化诊断。
- 如果问题和权限、网络、配置联动较深，仍然可能要配合其他技能一起使用。

## 安装方式

```bash
clawhub install openclaw-troubleshoot-cn
```
