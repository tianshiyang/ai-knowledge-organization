## 一句话定位

覆盖配置管理、模型设置、安全审计和 Gateway 管理的综合运维技能，适合已经进入稳定维护阶段的 OpenClaw 用户。

## 适合谁

- 已经长期使用 OpenClaw 并需要日常维护的人
- 同时负责配置、模型、渠道和安全审计的用户
- 需要统一处理 Gateway 管理和系统调优的人

## 典型使用场景

- 日常运维 OpenClaw 实例
- 配置管理和系统调优
- 安全审计需求
- Gateway 管理

## 为什么值得用

- 它覆盖的面比单点技能更广，适合作为平台专项的综合手册。
- 对已经进入“稳定运维”阶段的用户，这类技能比零散命令更能节省时间。

## 前端视角说明

对纯前端工作流帮助有限，但如果你是“既开发又维护工具环境”的那类用户，它会很实用。

## 通用性说明

不属于通用技能，主要服务 OpenClaw 平台维护。

## 风险与注意事项

- 覆盖范围广，使用前最好先明确自己是在做配置、审计还是排障。
- 对新手来说信息量偏大，建议先从更具体的问题型技能开始。

clawhub install openclaw-power-ops
