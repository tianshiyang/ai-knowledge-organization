# OpenClaw 技能整理

OpenClaw 是一个 AI Agent 框架，支持插件化扩展。本分类收录 ClawHub 上所有与 OpenClaw 相关的 Skills，按用途分为以下四个子类：

## 分类索引

### [运维类](./运维类/index.md)
故障排查、错误修复、运维防呆、CLI 操作与日常维护。
- openclaw-ops-guardrails
- openclaw-power-ops
- openclaw-error-fix
- openclaw-troubleshoot-cn

### [安全类](./安全类/index.md)
安全审计、服务器加固、Tailscale/Firewall 配置、策略检查。
- openclaw-server-secure-skill
- openclaw-security-policy-check

### [安装配置类](./安装配置类/index.md)
GCP 部署、CLI 操作、Owletto 插件安装配置、CLI 横幅屏蔽。
- openclaw-cli
- create-new-openclaw-in-gcp
- owletto-openclaw
- shut-up-openclaw-cli

### [其他](./其他/index.md)
暂未归类的工具型 Skills。
- （暂无）
