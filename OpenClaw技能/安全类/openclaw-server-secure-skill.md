## 一句话定位

一个把服务器加固、OpenClaw 安装和 Tailscale / Firewall 配置放到一起的安全导向技能。

## 适合谁

- 需要把 OpenClaw 部署到服务器的用户
- 在意网络安全和防火墙配置的人
- 同时负责安装与安全加固的维护者

## 典型使用场景

- 服务器安全加固需求
- OpenClaw Agent 安装配置
- Tailscale 网络配置
- Firewall 防火墙配置
- 安全性评估与加固

## 为什么值得用

- 它把“先装起来”和“装得更安全”放在同一条路径上，适合不想把部署和安全割裂开的用户。
- 对外网节点、远程机器和长期运行环境尤其有帮助。

## 前端视角说明

如果你只是在本地用 OpenClaw，可以跳过；如果你准备把它装到服务器，这页就会比一般教程更有价值。

## 通用性说明

它属于服务器和平台安全范畴，不是通用工作流技能。

## 风险与注意事项

- 涉及服务器、网络和防火墙配置，误操作成本高于普通文档型技能。
- 更适合有基础设施经验或愿意逐步验证配置的用户。

## 安装方式

```bash
clawhub install openclaw-server-secure-skill
```
