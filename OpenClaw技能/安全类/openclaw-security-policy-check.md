## 简介

自动对 OpenClaw Gateway 进行安全审计，通过检查关键配置设置来识别和报告常见安全风险。

## 适用场景

- 安全审计需求
- 合规性检查
- 发现潜在安全风险
- 定期安全评估

## 核心功能

- 配置安全检查
- 风险识别与报告
- 安全建议生成
- 常见安全问题排查

## 安装方式

```bash
clawhub install openclaw-security-policy-check
```
