# Claude Code 使用指南

Claude Code 是 Anthropic 推出的**终端原生 AI 编程代理**，采用「AI 驱动，你来监督」的模式。它能自主读取代码库、执行命令、编辑文件、运行测试，并在多文件间迭代解决问题。

## 为什么关注 Claude Code？

- **自主性强**：给出任务描述后，Claude Code 可独立完成探索→规划→实现→验证的完整闭环
- **上下文窗口巨大**：支持最高 100 万 token 的上下文，可以一次性理解大型代码库
- **Token 效率高**：比同类工具节省约 5.5 倍 token 消耗，返工率低 30%
- **开发者满意度领先**：2026 年调研中以 46% 的满意度远超竞品
- **行业影响力**：目前每日产出约 13.5 万个 GitHub commit（占公开 commit 的 ~4%），预计 2026 年底将超过 20%
